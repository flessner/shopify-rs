# Shopify Rust

This repository hosts two packages:
- [shopify_storefront](/shopify_storefront): Rust client for the [Shopify Storefront API](https://shopify.dev/docs/api/storefront/latest).
- [leptos_shopify](/leptos_shopify): Integration layer for [Leptos](https://github.com/leptos-rs/leptos).
